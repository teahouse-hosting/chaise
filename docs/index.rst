Chaise
======

Chaise is a rich client library for :external+couchdb:std:doc:`index`.

* Async
* Wraps APIs and provides useful abstractions
* Handles document serialization
* Tries to be only mildly opinionated
* Encourages best practices and mature CouchDB handling

This includes features like:

* :ref:`pooling`
* :ref:`types`
* :ref:`migrations`
